Multiple Sequence Alignment

Clustal Omega is a new multiple sequence alignment program that uses seeded guide trees and HMM profile-profile techniques to generate alignments between three or moresequences. For the alignment of two sequences please instead use our pairwise sequence alignment tools.

Important note:This tool can align up to 4000 sequences or a maximum file size of 4 MB.
